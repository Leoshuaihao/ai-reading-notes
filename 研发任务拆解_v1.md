## 三、里程碑 M2：AI对话API接入（2天）

### 任务 M2-T1：Cloudflare Workers API代理层

| 属性 | 内容 |
|------|------|
| **关联PRD** | F-05、AC-17、AC-18 |
| **关联评审** | H3、M3、T1、T2 |
| **预估工时** | 4h |

**子任务**：

- [ ] **M2-T1.1** 创建 Cloudflare Worker 项目，配置 wrangler.toml
- [ ] **M2-T1.2** 实现 `/api/chat` 端点：
  - 接收 `{ message, chapter_id, question_id, conversation_history }`
  - 转发到 AI API（OpenAI / Claude / 国产大模型）
  - 返回 SSE 流式响应
  - API Key 存储在 Worker 环境变量中
- [ ] **M2-T1.3** 实现速率限制：
  - IP 级别：每分钟最多 30 条
  - 前端配合：10 秒内最多 5 条（AC-18）
- [ ] **M2-T1.4** 实现错误处理与降级：
  - API 超时（5s）→ 返回 504 + 错误提示
  - API 返回错误 → 返回 502 + 错误提示
  - API 不可用 30 分钟 → 返回 503 + 维护提示（评审 T3）

### 任务 M2-T2：前端AI对话模块升级

| 属性 | 内容 |
|------|------|
| **关联PRD** | F-05-E1、F-05-E2、F-05-E3、F-05-U2、F-05-U3 |
| **关联评审** | M3 |
| **预估工时** | 4h |

**子任务**：

- [ ] **M2-T2.1** 替换模拟对话为真实 API 调用
  - 使用 `fetch` + `ReadableStream` 接收 SSE 流式响应
  - 实现打字机效果逐字显示
- [ ] **M2-T2.2** 实现对话状态管理：
  - 每道思考题的对话历史独立存储
  - localStorage key 格式：`chat_{book_id}_{chapter_id}_{question_id}`
  - 最多保留 30 轮对话（EX-13）
- [ ] **M2-T2.3** 实现输入框交互完善：
  - 发送后按钮 disabled，防抖 1 秒（EX-12）
  - 空消息拦截
  - AI 返回空响应时显示提示，不保留空气泡（EX-19）
- [ ] **M2-T2.4** 实现错误/加载/离线状态 UI（评审 M3）
  - 加载态：`AI正在思考...` 打字指示器
  - 超时态：5秒后提示"响应较慢，请稍候"（EX-03）
  - 错误态：显示"AI暂时无法回应，请稍后再试"（EX-04）
  - 离线态：发送按钮显示"离线"，点击提示"请检查网络连接"（EX-10）
- [ ] **M2-T2.5** 实现安全处理：
  - 用户输入 HTML 转义（防 XSS）（AC-17）
  - AI 回复中的原文引用使用引用块样式（F-05-E3）
- [ ] **M2-T2.6** 实现成本控制：
  - 前端 token 计数器（粗略估算）
  - 免费用户每日 20 轮限制提示（EX-11）

## 七、风险与阻塞项

| # | 风险 | 影响 | 缓解措施 |
|---|------|------|----------|
| R1 | AI API 选型未定（OpenAI vs Claude vs 国产） | 阻塞 M2 | 优先用 Claude API（长上下文优势），备选 DeepSeek |
| R2 | AI API 月度费用超预算 | 阻塞上线 | M2-T2.6 前端限制免费用户每日20轮；Worker 层 IP 限流 |

| R4 | Cloudflare 服务不可用（中国大陆访问） | 影响用户访问 | 备选：腾讯云静态托管 + 国内 AI API |
| R5 | 第二本书 EPUB 获取困难 | 不影响 M1/M2/M4 | 可先上线单本书，第二本书内容后补 |
