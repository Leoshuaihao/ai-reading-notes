Ⅰ · 估值

一家企业到底值多少钱？如何在价格与价值的缝隙中生存

概念 01

内在价值

企业真正值多少钱——独立于市场报价的价值之锚，一切估值的起点。

3 位大师的视角 ▼
格雷厄姆《证券分析》

内在价值是被事实证明的价值——资产、收益、股息，而非市场报价。

→ 深入阅读
巴菲特《致股东的信》

内在价值是企业余生现金流的折现值——账面价值告诉你付出了什么，内在价值告诉你得到什么。

→ 深入阅读
多尔西《巴菲特的护城河》

内在价值的持久性取决于护城河宽度。

→ 深入阅读
概念 02

安全边际

买价与价值之间的缓冲垫——为错误、厄运与未知留出余地。

3 位大师的视角 ▼
格雷厄姆《聪明的投资者》

安全边际是投资的中心概念——为错误、厄运、未知留出缓冲。

→ 深入阅读
卡拉曼《证券分析》导读

安全边际是价值投资唯一的"秘密"，公开了90年依然有效。

→ 深入阅读
塔勒布《反脆弱》

安全边际本质是负凸性的对冲——为未知的未知留出空间。

→ 深入阅读
哈格斯特朗《巴菲特之道》

巴菲特将安全边际内化为性格——不是计算出来的数字，而是决策时的本能。

→ 深入阅读
概念 03

市场先生

把市场当作一位情绪化的报价员——利用他的报价，而不是追随他的情绪。

3 位大师的视角 ▼
格雷厄姆《聪明的投资者》

市场先生是你的仆人，不是你的向导——利用他的报价，不受他的情绪感染。

→ 深入阅读
马克斯《投资最重要的事》

市场的钟摆永远在贪婪与恐惧间摆动，极端位置才是机会。

→ 深入阅读
利弗莫尔《股票大作手回忆录》

市场永远不会错，但意见常常会错——投机者读的就是市场先生的情绪。

→ 深入阅读

Ⅱ · 竞争与企业

什么样的企业值得长期持有？在哪里下注、如何看懂生意本身

概念 04

护城河

保护企业长期超额利润的结构性竞争优势——城堡值钱，是因为河够宽。

3 位大师的视角 ▼
巴菲特《致股东的信》

我们要的是被宽阔护城河保护的经济城堡。

→ 深入阅读
多尔西《巴菲特的护城河》

护城河四来源——无形资产、转换成本、网络效应、成本优势。

→ 深入阅读
费雪《怎样选择成长股》

卓越企业的护城河在于组织能力——研发、销售、管理层的持续进化。

→ 深入阅读
哈格斯特朗《巴菲特之道》

巴菲特的12条永恒原则中，护城河横跨企业准则和管理准则两组——识别它，然后信任管理层不挖穿它。

→ 深入阅读
概念 05

能力圈

只在自己真正理解的领域下注——圈的大小不重要，知道边界才重要。

3 位大师的视角 ▼
巴菲特《致股东的信》

能力圈的大小不重要，知道边界在哪里才重要。

→ 深入阅读
芒格《穷查理宝典》

如果你不知道自己能力圈的边界，你就没有能力圈。

→ 深入阅读
林奇《彼得·林奇的成功投资》

投资你了解的公司——你的职业和生活就是天然的能力圈。

→ 深入阅读
概念 06

成长股

寻找能持续成长十年的企业——但警惕为成长支付过高的价格。

3 位大师的视角 ▼
费雪《怎样选择成长股》

用15条原则找到能持续成长十年的企业，然后几乎永远不卖。

→ 深入阅读
林奇《彼得·林奇的成功投资》

快速增长型是十倍股的温床——但要区分真成长和伪成长。

→ 深入阅读
格雷厄姆《聪明的投资者》· 反方

为成长支付过高价格是投资者亏损的主要来源。

→ 深入阅读

Ⅲ · 风险与不确定性

风险不是波动，是你不知道的东西——以及周期与极端事件的逻辑

概念 07

风险

不是波动性，而是永久性损失的概率——以及你的模型之外的未知。

3 位大师的视角 ▼
马克斯《投资最重要的事》

风险不是波动性，是永久性损失的概率。

→ 深入阅读
塔勒布《黑天鹅》

真正的风险是你的模型里没有的东西——从未发生过的事。

→ 深入阅读
格雷厄姆《证券分析》

风险控制的起点是透彻分析——不懂的东西才是最大的风险。

→ 深入阅读
概念 08

周期

万物皆有周期——你无法预测未来，但可以知道自己身处何处。

3 位大师的视角 ▼
马克斯《周期》

你无法预测未来，但可以知道自己身处周期何处。

→ 深入阅读
索罗斯《金融炼金术》

周期不是自然现象，是认知与现实互相强化的反身性过程。

→ 深入阅读
利弗莫尔《股票大作手回忆录》

华尔街没有新鲜事——周期背后是不变的人性。

→ 深入阅读
概念 09

黑天鹅与肥尾

历史由极端事件塑造——既然无法预测，就构建能从中获益的结构。

3 位大师的视角 ▼
塔勒布《黑天鹅》

历史由极端事件塑造，钟形曲线是伟大的知识骗局。

→ 深入阅读
塔勒布《反脆弱》

既然无法预测黑天鹅，就构建能从中获益的结构——杠铃策略。

马克斯《周期》· 呼应

周期极端处，就是黑天鹅的温床——信贷周期见顶时最脆弱。

Ⅳ · 心理与行为

投资者最大的敌人是自己——偏误、逆向与仓位的艺术

概念 10

人类误判心理学

投资者最大的敌人不是市场，是自己的心理偏误——尤其当多个偏误共振时。

3 位大师的视角 ▼
芒格《穷查理宝典》

25种心理倾向叠加产生lollapalooza效应——多个偏误同向共振时人会彻底失去理性。

→ 深入阅读
塔勒布《随机漫步的傻瓜》

幸存者偏差让我们只看到赢家——大部分成功是被随机性愚弄。

→ 深入阅读
格雷厄姆《聪明的投资者》

投资者最大的敌人不是市场，是自己。

→ 深入阅读
概念 11

逆向投资

在众人恐惧处买入，在众人贪婪处警惕——但逆向必须以正确为前提。

3 位大师的视角 ▼
马克斯《投资最重要的事》

在别人沮丧地卖出时买入——逆向的三个条件：不同、正确、敢行动。

→ 深入阅读
巴菲特《致股东的信》

别人贪婪时恐惧，别人恐惧时贪婪。

→ 深入阅读
林奇《战胜华尔街》

最好的股票藏在最被冷落的行业——沙漠之花。

→ 深入阅读

过度分散是对无知的对冲——真正了解的公司不需要很多。
