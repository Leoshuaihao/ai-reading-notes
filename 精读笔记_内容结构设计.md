# 《Common Stocks and Uncommon Profits》精读笔记 — 内容结构设计

## 全局结构

```
书籍首页
├── 书籍概览卡片（封面、作者、一句话简介、阅读难度、预计阅读时间）
├── 为什么这本书值得精读
├── 作者背景：Philip Fisher 是谁
├── 阅读路线图（全书结构导航）
└── 开始阅读 →
```

## 每章结构

```
第N章：[章节标题]
├── 📍 章节定位（本章在全书中解决什么问题，承上启下）
├── 📖 原文金句（1-3句，英文原文 + AI精准翻译 + 为什么这句话重要）
├── 🧠 核心概念深度解读
│   ├── 概念1：名称 + 原文引用 + 通俗解释 + 为什么中文读者容易误解
│   ├── 概念2：...
│   └── ...
├── 🌍 文化背景补充
│   ├── 书中提及的公司/人物/事件（如提及1950年代的德州仪器等）
│   └── 当时的市场环境与现在的关联
├── ⚔️ 反方观点与辩论
│   ├── 哪些投资者/学者不同意Fisher的这个观点？
│   └── 他们的理由是什么？
├── 🇨🇳 中国市场关联
│   ├── Fisher的这个方法在中国市场适用吗？
│   └── A股/港股案例类比
├── 📊 关键术语词典（本章出现的专业术语中英对照）
├── 💡 本章要点总结（3-5条可执行的takeaway）
├── 🤔 思考题（2-3个问题，帮助读者内化）
└── 💬 读者讨论区入口
```
